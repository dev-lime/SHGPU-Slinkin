ЛР3. Управлением холстом.

Задание

Разработать приложение для построения графиков функций с учетом возможностей, представленных в примере.

Основные компоненты, используемые в примере:
TPaintBox - для построения графика
TPSScript - для рассчета формул

Вспомогательные компоненты, используемые в примере:
TIpHtmlPanel - для интерактивной html-помощи
TFloatSpinEdit - для полей масштабирования
TColorBox - для выбора цвета графика
TEdit - для ввода формулы
TLabel - для информационных полей
TButton - для кнопки "Рисовать"
TPanel - для группировки компонентов

Студент имеет право использовать вместо указанных другие компоненты, входящие в базовый состав среды программирования.

Основные требования к приложению

  • корректное масштабирование как графика, так и основного окна приложения
  • реализация возможностей, описанных в интерактивной помощи
После защиты основного задания предлагаются независимые друг от друга бонусы:

+30 баллов первым трем студентам - за дополнительную поддержку графиков на базе параметрических уравнений.
+50 баллов первым двум студентам - за дополнительную поддержку нескольких графиков функций.
+100 баллов первому студенту - за дополнительную поддержку трехмерных графиков.
