ЛР3. Функции и макроопределения.

Создаваемые функции и макроопределения не должны обращаться к внешним для них переменным.

Задания

  1. (30) Решить задачи 47аб, 68а, с помощью функций и макроопределений, предназначенных для использования в выражениях.
  2. (30) Решить задачи 466,467,469 (равномерное распределение пробелов) с помощью функций типа void.
  3. (20-40) Решить задачи 466,467,469 (равномерное распределение пробелов) с помощью макроопределений. Полученные макроопределения должны вызываться как процедуры и не могут использоваться в выражениях.
